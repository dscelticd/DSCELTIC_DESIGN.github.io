새 업무를 등록하려면 아래 버튼을 눌러 GitHub 이슈를 생성하세요. 이슈 양식에는 JSON 코드 블록으로 업무 정보를 입력해야 합니다. 예시는 script.js 파일 상단 주석을 참고하세요.
